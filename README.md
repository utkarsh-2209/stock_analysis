# StockLens: Indian Stock Market Analyzer

StockLens is a full-stack web application designed to help investors analyze Indian stocks using real-time fundamental data from Yahoo Finance, accompanied by advanced AI-driven contextual reviews generated by Google Gemini 2.5 Flash.

## 🌟 Features

- **Real-Time Fundamentals**: Secure key metrics like P/E Ratio, Market Cap, ROE, Debt/Equity, and recent news.
- **AI Stock Reviews**: Get comprehensive analysis on Quality, Valuation, Financial Health, and Sentiment with investment verdicts (Buy/Hold/Sell/Avoid).
- **TradingView Integration**: View fully interactive embedded charts for deep technical analysis on any listed Indian stock.
- **AI Chart Predictions**: Take a screenshot of your TradingView chart, upload it, and let Gemini detect patterns, trends, support/resistance levels, and outline risk-to-reward ratios.
- **Historical Timelines**: Quickly retrieve and map historical price actions (1D, 1M, 6M, YTD, 1Y, 5Y).

## 🛠️ Tech Stack

- **Frontend**: Custom HTML/Vanilla JS wrapped in [Vite](https://vitejs.dev/) for blazing fast module bundling.
- **Backend**: [Node.js](https://nodejs.org/) & [Express.js](https://expressjs.com/) REST API.
- **APIs**: 
  - `yahoo-finance2` for fundamental data and current quotes.
  - `@google/generative-ai` to power the expert AI analysis and chart insight models.

## 🚀 Getting Started Locally

### Prerequisites
- Node.js `18+`
- A valid **Google Gemini API Key** (You can retrieve yours from [Google AI Studio](https://aistudio.google.com/)).

### Installation

1. Clone this repository (or download the source code).
2. Install all dependencies for both the frontend and backend simultaneously from the root directory:
   ```bash
   npm run install:all
   ```

### Running the App

Start both the Vite frontend server and the Express backend concurrently:
```bash
npm run dev
```

- **Frontend**: http://localhost:5173
- **Backend API**: http://127.0.0.1:3001

> **Note:** The very first time you launch the Web App, head to the **Settings** page via the top navigation bar and enter your Gemini API Key to unlock all AI Analysis capabilities.

## 🌐 Deploying to Vercel

The repository is natively configured to run on Vercel as a single project using Vercel's Serverless environment.

1. Push your repository to GitHub.
2. Go to your [Vercel Dashboard](https://vercel.com/new) and click **Import Project**.
3. Select your repository. 
4. Leave the **Framework Preset** as Vite and **Root Directory** as `./` (the root).
5. In the **Build and Output Settings** section, securely override the defaults to match the custom directories:
   - **Build Command**: `npm run install:all && cd client && npm run build`
   - **Output Directory**: `client/dist`
6. Click **Deploy**. Vercel will automatically compile your React frontend and expose your Express Server functions via the `api/index.js` wrapper!
